# History

The **Scientific Grid Computing and Cloud Computing Resources for Community (SGCCRfC)** initiative began in **2013**, during a period of rapid transformation in scientific computing.

---

## Origins (2013)

The project was founded to commemorate and advance breakthroughs in:

- Cloud computing  
- Grid computing  
- Distributed scientific workflows  
- HPC infrastructure evolution  

Researchers at the time struggled to access unified, well-organized resources. SGCCRfC was created to **bridge that gap**.
