# SAR / InSAR

SAR (Synthetic Aperture Radar) and InSAR (Interferometric SAR) are active microwave remote sensing techniques used for:

- Deformation mapping  
- Subsidence monitoring  
- Volcanic activity  
- Glacier dynamics  
- Landslides
